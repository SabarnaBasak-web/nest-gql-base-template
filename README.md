## Description

base template for backend project setup using nestjs graphql and postgres with typeorm

- typeorm setup
- basic example of query and mutation using a table connected to example DB

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
